Documentation

dostlang is dynamically typed toy programming language, written in Python.

General

shuru is the entrypoint for the program and all program must end with khatam. Anything outside of it will be ignored.

    # Write code here

Variables

Variables can be declared using rakho.

shuru
    rakho a = 10;
    rakho b = "two";
    rakho c = 15;
    a = a + 1;
    b = "twenty one";
    c = c * 2;
khatam

Types

Numbers and strings are like other languages. sach and jhoot are the boolean values.

shuru
    rakho a = 10;
    rakho b = 10 + (15*20);
    rakho c = "two";
    rakho d = 'ok';
    rakho f = sach;
    rakho g = jhoot;
khatam

Built-ins

Use dikhao to print anything to console.

shuru
    dikhao("Hello World");
    rakho a = 10;
    {
        rakho b = 20;
        dikhao(a + b);
    }
    dikhao(5, 'ok', sach, jhoot);
khatam

Conditionals

DostLang supports conditional constructs. The agar block will execute if condition is sach, otherwise subsequently added nested warna { agar ... } blocks will execute if their condition is sach, and the fallback warna block will execute if all of the above conditions are jhoot.

shuru
    rakho a = 10;
    agar (a < 20) {
        dikhao("a is less than 20");
    } warna {
        agar (a < 25) {
            dikhao("a is less than 25");
        } warna {
            dikhao("a is greater than or equal to 25");
        }
    }
khatam

Loops

Statements inside jabtak blocks are executed as long as a specified condition evaluates to sach. If the condition becomes jhoot, statements within the loop stop executing.

shuru
    rakho a = 0;
    jabtak (a < 10) {
        a = a + 1;
        dikhao(a);
    }
    dikhao("done");
khatam
